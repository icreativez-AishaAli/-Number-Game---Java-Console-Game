🎮 Number Game - Java Console Game

A fun, interactive number guessing game built in Java as part of the DecodeLabs Industrial Training Kit (Batch 2026). Test your intuition and see how well you can guess random numbers!

 Features
-  Random number generation (1-100)
-  Multiple rounds support
-  Limited attempts per round (7 attempts)
-  Dynamic scoring system based on attempts remaining
-  Final score tracking across all rounds
-  Real-time feedback (Too high/Too low)
-  Colorful console output
- Input validation

  
-  How to Play
1. The computer randomly selects a number between 1 and 100
2. You have 7 attempts to guess the correct number
3. After each guess, you'll receive hints:
   -  "Too low!" - Guess a higher number
   -  "Too high!" - Guess a lower number
4. Score points based on how quickly you guess correctly
5. Play multiple rounds to accumulate a total score
6. Try to achieve the highest possible score!
